# [Smart Attendance System](https://attendance-app-8dkqufve4pz4ryxbz9xvga.streamlit.app/)

A web-based student attendance system built with Python and Streamlit. This application replaces manual paper attendance with a secure, digital QR Code workflow. It includes role-based access for Students, Lecturers, and Administrators.

---

## Key Features

* **Role-Based Login:** Secure authentication with separate dashboards for Students, Lecturers, and Admins.
* **QR Code Attendance:** Lecturers generate unique session codes; students scan to mark present instantly.
* **Automatic Analytics:** The system automatically calculates if a student meets the **75% attendance requirement** for exams.
* **Real-Time Reports:** Lecturers can view attendance history and download logs as CSV files.
* **Admin Management:** Full control to register new users (Students/Lecturers) and assign departments.
* **Device Independent:** Works on any device with a browser (Laptop, Phone, Tablet).
